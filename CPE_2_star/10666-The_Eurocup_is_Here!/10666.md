## 比賽規則

我們是歐洲盃的冠軍！好吧……差不多是。實際上，我們在第一輪就輸了，一場比賽也沒贏。但我們是被真正的冠軍打敗的，所以樂觀一點想，我們可能是第二名！盃賽總是這麼不公平……

讓我們考慮一個包含 $N$ 輪的盃賽，共有 $2^N$ 支隊伍參加。每一輪的獲勝者晉級下一輪。

- **第一輪**：有 $2^N$ 支隊伍。
- **第二輪**：有 $2^{N-1}$ 支隊伍。
- 以此類推。
    
隊伍編號為 $0, 1, 2, \dots, 2^N - 1$。每一輪的比賽順序皆依照編號排列。假設兩支隊伍比賽時，編號較小的那一隊總是獲勝。

結論是：

- **第一輪對戰組合**：(0 vs 1), (2 vs 3), (4 vs 5) ...    
- 第二輪對戰組合：(0 vs 2), (4 vs 6) ...
- （圖片展示了 $N=3$ 時的情況，編號 0 最終成為冠軍。）

## 排名規則

如果隊伍 $X$ 擊敗了隊伍 $Y$，則稱 $X$ **優於（better than）** $Y$；如果 $X$ 輸給了 $Z$，則 $X$ **劣於（worse than）** $Z$。此外，這種關係具有傳遞性：

- 如果 $X$ 優於 $Y$，且 $Y$ 優於 $Z$，則 $X$ 優於 $Z$。
- 例如，在 $N=3$ 的情況下，隊伍 7 劣於隊伍 6、隊伍 4 和隊伍 0。

比賽結束後，所有隊伍會被賦予 $1$ 到 $2^N$ 的**排名（Ranking）**（每隊排名皆不同，冠軍為 1，亞軍為 2，依此類推）。隊伍 $X$ 的排名記為 $R(X)$。

一個**有效的排名（Valid Ranking）**必須符合以下性質：

> **如果 $X$ 優於 $Y$，則 $R(X) < R(Y)$。**

顯然，存在許多種可能的有效排名（因為某些隊伍之間沒有直接或間接的勝負關係）。

- **樂觀排名（Optimistic classification）**：在所有可能的有效排名中，$R(X)$ 的**最小值**。
- **悲觀排名（Pessimistic classification）**：在所有可能的有效排名中，$R(X)$ 的**最大值**。

給定總輪數 $N$ 和隊伍編號 $X$，請計算該隊伍的樂觀與悲觀排名。

---

## 輸入格式 (Input)

輸入的第一行包含一個整數 $M$，表示測試資料的組數。

每組測試資料包含兩個整數 $N$ 和 $X$。

- $0 < N < 31$（總輪數） 
- $0 \le X \le 2^N - 1$（隊伍編號）

## 輸出格式 (Output)

對於每組測試資料，輸出一行包含兩個整數：該隊伍的**樂觀排名**與**悲觀排名**。

# 題目大意

## 舉例

比如說

如果 `N = 3, X = 1`： team 1 會在 round 1 就輸給 team 0，代表：

- 樂觀排名：我只輸給第一名 (team 0 必拿冠軍) ，所以我是第二名
- 悲觀排名：我**沒有擊敗過任何隊伍**，那我肯定是最爛的，所以我是第八名

---

如果 `N = 4, X = 10`： team 10 會贏 round 1，輸 round 2 (team 8) ，代表：

- 樂觀排名：我只輸給 team 8 (team 0 必拿冠軍) ，team 8 的排名又輸給 team 0 ，所以我是第三名
- 悲觀排名：我**只擊敗過一支隊伍**，那我肯定是倒數第二爛的，所以我是第十五名

## 歸納

聰明的你肯定會發現，奇數隊伍 round 1 必輸；team 0必勝。而每一輪都是二進制間的關係。我們將圖片中的隊伍編號轉換成二進制：



實在太神奇了，只需要從左到右判斷有幾個「連續的0、第一次出現1」。

|                              | **意義**               | **N= 4, X=10 (1010) 的計算**                   |
| ---------------------------- | -------------------- | ------------------------------------------- |
| **末尾連續的 0 個數 (win)**         | 你贏的場數                | `101`**`0`** $\rightarrow$ 末尾有 1 個 0，贏 1 場。 |
| 第一次二進位出現 1 之後，**到頭總共 1 的個數** | 每多一個 1 代表後續回合會在輸一次   | **`1`**`0`**`1`**`0` $\rightarrow$ 有 2 個 1。 |
|                              |                      |                                             |
| **樂觀排名**                     | $1 + (\text{1 的個數})$ | $1 + 2 =$ **3**                             |
| **悲觀排名**                     | $2^N - 2^W + 1$      | $2^4 - 2^1 + 1 = 16 - 2 + 1 =$ **15**       |
