# 題解

首先必須找規律，不外乎就是排列組合、遞迴

看下圖還真的是遞迴，而且是 Fibonacci Sequence


$$
a_{n} = a_{n-1} + a_{n-2}, \quad n\geq 2, \quad a_{0} =1, a_{1} = 1
$$


# Code

```cpp
#include <cstdio>

int main()
{
    int n;
    
    while (scanf("%d", &n) == 1 && n != 0) 
    {
        long long a = 1, b = 1;
        
        for (int i = 2; i <= n; i++) 
        {
            long long temp = a + b;
            a = b;
            b = temp;
        }
        
        printf("%lld\n", b);
    }
    
    return 0;
}
```
