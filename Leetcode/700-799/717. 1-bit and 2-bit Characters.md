# 題解（破題）

> 你得想到或注意到

```
遇到 0 → 只能是 1-bit character，跳 1 格
遇到 1 → 只能是 2-bit character，跳 2 格
```

因此**從左到右只有唯一的解碼方式**

- 看最後跳到哪裡，如果剛剛好最後遇到 0，那就是 `1-bit`。

```
[1, 0, 0]        [1, 1, 1, 0]
 ↑                ↑
 i=0, 遇1 → +2    i=0, 遇1 → +2
       ↑                ↑
       i=2, 停止         i=2, 遇1 → +2
                               ↑
                               i=4, 停止（越過 n-1）
                               
i==n-1 → true    i!=n-1 → false
```

# Code

```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
	bool isOneBitCharacter(vector<int>& bits)
	{
	    int n = bits.size();
	    int i = 0;
	    while (i < n - 1)  { i += bits[i] + 1; }
	    return i == n - 1;
	}
};

int main()
{
    vector<int> bits1 = {1, 0, 0};
    vector<int> bits2 = {1, 1, 1, 1, 0, 1, 0};

    Solution sol;
    println("{}", sol.isOneBitCharacter(bits1));  // true
    println("{}", sol.isOneBitCharacter(bits2));  // false

    return 0;
}
```
