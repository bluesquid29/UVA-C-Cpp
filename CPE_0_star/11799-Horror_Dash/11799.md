# 題目大意

題目**落落長**，但其實很簡單。

> Given the speed of every scary creature, you are to find out the minimum speed that the clown must maintain so as not to get caught even if they keep on running forever.

既然要不被追到，那就代表 clown 的速度要**大於等於** scary creatures。白話文就是：找最大值。

# IO

```
3
2 1 3
4 5 6 7 8
6 1 9 77 9 5 0
```

```
Case 1: 3
Case 2: 8
Case 3: 77
```

# Code

```cpp
#include <cstdio>

int main() 
{
	int T;
	scanf("%d", &T);
	
	for (int i = 1; i <= T; i++)
	{
		int N;
		scanf("%d", &N);
		int max = 0;
		
		while(N--) 
		{
			int creature;
			scanf("%d", &creature);
			if(max < creature) max = creature;
		}
		
		printf("Case %d: %d\n", i, max);
	}
    return 0;
}
```
