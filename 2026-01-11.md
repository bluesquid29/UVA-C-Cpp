# C++ 程式碼撰寫風格指南

## 基本設定
- 使用 `#include <bits/stdc++.h>` 和 `using namespace std;`（競賽/練習用途）
- 目標：CPE、UVa 線上評測系統

## 程式碼風格

### Header 與 Namespace

```cpp
#include <bits/stdc++.h>
using namespace std;
```

### 大括號規則

- **單行敘述**：省略大括號
- **多行敘述**：使用大括號
- 範例：

```cpp
for (int i = 0; i < n; i++)
    for (int j = 0; j < m; j++)
        if (arr[i][j] > 0)
        {
            sum += arr[i][j];
            count++;
        }
```

### I/O 優化

大量輸入輸出時使用：

```cpp
ios_base::sync_with_stdio(false);
cin.tie(nullptr);
```

⚠️ 不可與 `printf`/`scanf` 混用

### 其他偏好

- 使用 `'\n'` 取代 `endl`
- 變數命名清晰簡潔（如 `cnt`, `L`, `train`）
- 使用 STL 容器（`vector`, `map`, `set` 等）

## 程式碼範例參考

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int N;
    cin >> N;
    while (N--)
    {
        int L;
        cin >> L;
        vector<int> train(L);  
        
        for (int i = 0; i < L; i++)
            cin >> train[i];
        
        // Bubble sort
        int cnt = 0;
        for (int i = L - 1; i > 0; i--)
            for (int j = 0; j < i; j++)
                if (train[j] > train[j + 1])
                {
                    swap(train[j], train[j + 1]);
                    cnt++;                   
                }
        
        cout << "Optimal train swapping takes " << cnt << " swaps.\n";
    }
    
    return 0;
}
```

## 注意事項

- 程式碼簡潔實用，避免過度工程化
- 程式碼中的註解使用英文，以清楚為主。
- 優先使用 C++ STL 而非純 C 寫法（除非效能考量）
- 命名風格：小寫+底線（如 `train_count`）
	- 大寫+底線（如 `Find_A`）表示 函數
