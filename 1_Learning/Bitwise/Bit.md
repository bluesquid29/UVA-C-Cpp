# Mask 遮罩

## 說明

在程式設計或電腦科學裡，「mask 遮罩」通常是指一個二進位的數字，用來在位元運算中選擇性地保留、清除或翻轉某些位元。它就像一個「濾鏡」或「模板」，決定哪些位元要被操作、哪些位元要保持不變。

應用方式有：

- **保留 (Extracting)**
- **清除 (Clearing)**
- **翻轉 (Toggling)**
- **設定 (Setting)**

## 舉例

```cpp
unsigned a = 151;          
unsigned b = (1 << 5) - 1;
```

#### 延伸 1：保留所有位數

1. **初始狀態**：`mask` 從 `1` (`0...0001`) 開始。
2. **判斷條件**：`while((mask & n) != n)`。這意味著只要 $n$ 當中還有任何一個位元是 `mask` 沒蓋到的（即 `mask` 的範圍還不夠大），就繼續執行。
3. **擴張遮罩**：
    - `mask = mask << 1;`：將既有遮罩左移一位。
    - `mask += 1;`：將最右邊補上 1。
    - 這兩步結合起來效果就是把遮罩從 `1` 變成 `11` -> `111` -> `1111` ... 依此類推
