# 11365 - Copying DNA

Evolution is a seemingly random process which works in a way which resembles certain approaches we use to get approximate solutions to hard combinatorial problems. You are now to do something completely different.

Given a DNA string $S$ from the alphabet {**A, C, G, T**}, <mark>find the minimal number of copy operations needed to create another string</mark> $T$. 

- You may reverse the strings you copy, and copy both from $S$ and the pieces of your partial $T$. 
- You may put these pieces together at any time. 
- You may only copy contiguous parts of your partial $T$, and all copied strings must be used in your final $T$. 

Example: From $S$ = “ACTG” create $T$ = “**GTACTATTATA**”

1. Get **GT.........** by copying and reversing “TG” from $S$.
2. Get **GTAC.......** by copying “AC” from $S$.    
3. Get **GTAC...TA..** by copying “TA” from the partial $T$.
4. Get **GTAC...TAAT** by copying and reversing “TA” from the partial $T$.
5. Get **GTACAATTAAT** by copying “AAT” from the partial $T$.

---

## Input

The first line of input gives a single integer, $1 \le t \le 100$, the number of test cases. 

Then follow, for each test case, 

- a line with the string $S$ of length $1 \le m \le 18$, 
- and a line with the string $T$ of length $1 \le n \le 18$.

## Output

- Output for each test case the number of copy operations needed to create $T$ from $S$,
- or “impossible” if it cannot be done.

---

## Sample Input

```
5
ACGT
GTAC
A
C
ACGT
TGCA
ACGT
TCGATCGA
A
AAAAAAAAAAAAAAAAAA
```

## Sample Output

```
2
impossible
1
4
6
```

# 中文翻譯

演化是一個看似隨機的過程，其運作方式類似於我們用來求解困難組合問題近似解的方法。不過現在你要做的是完全不同的事情。

給定一個來自字母集合 {**A, C, G, T**} 的 DNA 字串 $S$，請找出建立另一個字串 $T$ 所需的<mark>最少複製操作次數</mark>。

- 你可以將所複製的字串反轉，也可以從 $S$ 或是目前已經建立的部分 $T$ 中進行複製。
- 你可以在任何時候將這些片段組合起來。
- 你只能從目前的部分 $T$ 中複製連續子字串，且所有複製得到的字串都必須被使用在最終的 $T$ 中。

例如：由 $S$ =「ACTG」建立 $T$ =「**GTACTATTATA**」

1. 透過從 $S$ 複製並反轉「TG」，得到 **GT.........**
2. 透過從 $S$ 複製「AC」，得到 **GTAC.......**
3. 透過從目前的部分 $T$ 複製「TA」，得到 **GTAC...TA..**
4. 透過從目前的部分 $T$ 複製並反轉「TA」，得到 **GTAC...TAAT**
5. 透過從目前的部分 $T$ 複製「AAT」，得到 **GTACAATTAAT**

---

## 輸入

第一行為整數 $1 \le t \le 100$，表示測資的筆數。

接下來對於每筆測資：

- 一行字串 $S$，其長度為 $1 \le m \le 18$
- 一行字串 $T$，其長度為 $1 \le n \le 18$

---

## 輸出

- 對於每筆測資，請輸出從 $S$ 建立 $T$ 所需的最少複製操作次數；
- 若無法完成，請輸出「impossible」。
