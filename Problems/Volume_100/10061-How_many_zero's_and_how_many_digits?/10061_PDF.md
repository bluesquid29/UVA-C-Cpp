# 10061 - How many zero's and how many digits?

Given a decimal integer number you will have to find out how many trailing zeros will be there in its factorial in a given number system and also you will have to find how many digits will its factorial have in a given number system? You can assume that for a $b$ based number system there are $b$ different symbols to denote values ranging from $0 \dots b-1$. As for example $5! = 120$ (in decimal) but it is $78$ in hexadecimal number system. So in Hexadecimal $5!$ has no trailing zeros.

---

## Input

There will be several lines of input. Each line makes a block. Each line will contain a decimal number $N$ (a 20-bit unsigned number) and a decimal number $B$ ($1 < B \le 800$), which is the base of the number system you have to consider.

## Output

For each line of input output in a single line how many trailing zeros will the factorial of that number have in the given number system and also how many digits will the factorial of that number have in that given number system. Separate these two numbers with a single space. You can be sure that the number of trailing zeros or the number of digits will not be greater than $2^{31}-1$.

---

## Sample Input

```
2 10
5 16
5 10
```

## Sample Output

```
0 1
0 2
1 3
```

# 中文翻譯

給定一個十進制整數，你必須找在給定的進位制下，該數階乘（factorial）的末尾有多少個零 ；同時，你也必須找出該階乘在給定進位制下總共有多少位數 。你可以假設對於 $b$ 進位制，有 $b$ 個不同的符號來表示 $0 \dots b-1$ 的值 。舉例來說，$5! = 120$（十進制），但在十六進制中它是 $78$ 。因此，在十六進制下，$5!$ 的末尾沒有零 。

## Input

輸入包含多行，每一行代表一組測試資料 。每行包含一個十進制數 $N$（20 位元無號整數）以及一個十進制數 $B$ ($1 < B \le 800$)，後者即為你需考慮的進位制基底 。

## Output

對於每行輸入，請在一行中輸出該數階乘在指定進位制下的末尾零個數，以及該階乘在該進位制下的總位數 。這兩個數字之間請用一個空格隔開 。你可以確定末尾零的個數或總位數都不會超過 $2^{31}-1$ 。
