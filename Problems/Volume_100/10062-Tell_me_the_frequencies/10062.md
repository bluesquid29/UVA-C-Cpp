# 題解

輸入的字串 $\leq$ 1000，用 `for` 統計次數

```cpp
int cnt[129] = {0};
for (char c : line)  
{
	cnt[c]++;
}
```

---

假設輸入是 `"AAB"`：

```cpp
// 初始狀態
cnt[0~128] = {0, 0, 0, ..., 0}

// 讀到 'A' (ASCII 65)
line[0] = 'A' → (int)'A' = 65
cnt[65]++ → cnt[65] = 1

// 再讀到 'A'
line[1] = 'A' → (int)'A' = 65
cnt[65]++ → cnt[65] = 2

// 讀到 'B' (ASCII 66)
line[2] = 'B' → (int)'B' = 66
cnt[66]++ → cnt[66] = 1

// 結果：
cnt[65] = 2  // 'A' 出現 2 次
cnt[66] = 1  // 'B' 出現 1 次
```
