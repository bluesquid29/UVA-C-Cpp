# 題解

C++

```cpp
std::string input;
int cnt = 0;

while (std::cin >> input && input != "#")
{
	std::cout << "Case " << ++cnt << ": ";
	
	if (input == "ZDRAVSTVUJTE") 
		std::cout << "RUSSIAN";
```

C

```c
char input[14];
int cnt = 0;

while (scanf("%s", input) == 1 && strcmp(input, "#") != 0)
{
	printf("Case %d: ", ++cnt);
	
	if (strcmp(input, "ZDRAVSTVUJTE") == 0)
	{
		printf("RUSSIAN");
	}
```
