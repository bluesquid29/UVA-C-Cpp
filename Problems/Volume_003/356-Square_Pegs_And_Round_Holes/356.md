## 1. 垂直掃描

討論兩種情況：**完整 complete** 與**部份 segment** 掃描

```cpp
int complete = 0;
int segment = 0;

for (int i = 0; i < n; ++i)
{
	complete += floor(y[i+1]);
	segment += ceil(y[i]) - floor(y[i+1]);
}
```
