# 翻譯

在我們大學裡有一個名叫 mabu 的人，負責在走廊中開關燈。走廊裡的每一顆燈泡都有自己專屬的切換開關，也就是說，按一次燈會亮，再按一次燈就會熄。

為了節省電力（或者他可能有點瘋，或是其他原因），他做了一件很奇怪的事情。假設走廊裡有 n 顆燈泡，他會在走廊中來回走 n 次。在第 i 次行走時，他只會切換那些編號可以被 i 整除的燈泡開關。

當他走回起點時，不會再按任何開關。所謂的第 i 次行走，定義為：沿著走廊走一趟（並進行上述操作），然後再走回來（不做任何事）。

現在，你必須判斷：最後一顆燈泡的最終狀態是亮著（On）還是熄滅（Off）？

# 題解

找個例子 12 ，

$$
\large
\begin{aligned}
12 
&= 1 \cdot 12 \\
&= 2 \cdot 6 \\
&= 3 \cdot 4 \\
\end{aligned}
$$

所以最結果是：關燈。

---

下一個例子 25 ，

$$
\large
\begin{aligned}
25
&= 1 \cdot 25 \\
&= 5^{2}
\end{aligned}
$$

所以最結果是：開燈。

---

n 的因數要為**奇數個**，因此能滿足此條件的 n 只能是**完全平方數**。用程式語言判斷是否為**完全平方數**的方法，就是先開根號，再平方。（因為開根號只保留整數）

注意請使用 long long

---

# Code

```cpp
#include <cstdio>
#include <cmath>
 
int main()
{
    long long n, m;
    while(scanf("%lld", &n) == 1) 
    {
    	if (n == 0) break;
            
        m = (long long)sqrt(n);
 
        if (m * m == n)
        {
        	printf("yes\n");
        }
        else
        {
        	printf("no\n");
        }
    }
 
    return 0;
}
```
