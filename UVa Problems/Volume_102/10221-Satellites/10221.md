# 題解

> [!warning]
> 題目並沒有規定角度 $a$ 的範圍。因此為了萬無一失，我們必須考慮角度有可能超過 360 度，或是為負值。

# Code

```cpp
#include <iostream>
#include <string>
#include <iomanip>
#include <cmath>
using namespace std;

double const PI = acos(-1);

int main()
{
	cout << fixed << setprecision(6);
	
	double s;
	double a; 
	string angle;
	
	while (cin >> s >> a >> angle)
	{	
	    // 1. 處理單位：如果是分，先轉成度
	    if (angle == "min")  { a /= 60.0; }
	
	    // 2. 處理正負與超過一圈：取絕對值並對 360 取餘數
	    a = fmod(abs(a), 360.0);
	
	    // 3. 處理最短路徑：如果夾角大於 180，取其對應的小弧
	    if (a > 180.0)  { a = 360.0 - a; }
	
	    // 4. 「度」轉換成「弧度」
		double rad = PI * a / 180;
		
		// 計算題目要求
		double arc =       (6440 + s) * rad;
		double chord = 2 * (6440 + s) * sin(rad / 2);
		
		cout << arc << " " << chord << endl;
	}
	
	return 0;
}
```
