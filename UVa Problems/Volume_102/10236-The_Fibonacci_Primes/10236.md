# 題解

> [!warning]

確實，難點層層疊加：

**數學性質不直覺**

- 題目說「對所有較小 Fibonacci 數互質」，但實際要推導出「索引必須是質數（或4）」需要用到 $\gcd(F_m, F_n) = F_{\gcd(m,n)}$ 這個不太基本的恆等式
- 沒有這條性質就只能暴力，但 $N \le 22000$ 代表索引可能高達 ~25萬，$F_{250000}$ 根本沒法存

**數值處理**

- 大 Fibonacci 數字無法直接計算，要改用對數取前9位
- `double` 精度不夠，要用 `long double`，而且即使如此在極大 $n$ 時仍需謹慎

**邊界特例**

- $n = 4$ 不是質數但要特判，很容易漏掉
- $n = 1, 2$ 對應 $F_1 = F_2 = 1$，要排除

**考點組合**

> 篩法 + 數論恆等式 + 對數取位數 = 三個獨立知識點缺一不可

放在 UVa 上算是偽裝成實作題的數學題。
