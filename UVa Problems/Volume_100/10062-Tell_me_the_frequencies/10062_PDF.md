# 10062 - Tell me the frequencies!

Given a line of text you will have to find out the frequencies of the ASCII characters present in it. The given lines will contain none of the first 32 or last 128 ASCII characters. Of course lines may end with `\n` and `\r` but always keep those out of consideration.

---

## Input

Several lines of text are given as input. Each line of text is considered as a single input. Maximum length of each line is 1000. Input is terminated by end of file.

## Output

Print the ASCII value of the ASCII characters which are present and their frequency according to the given format below. A blank line should separate each set of output. Print the ASCII characters in the ascending order of their frequencies. If two characters are present the same time print the information of the ASCII character with higher ASCII value first.

---

## Sample Input

```
AAABBC
122333
```

## Sample Output

```
67 1
66 2
65 3

49 1
50 2
51 3
```

# 中文翻譯

給定一行文字，你必須找出其中出現的 ASCII 字元的出現頻率 。給定的文字行不會包含 ASCII 前 32 個字元或最後 128 個字元 。當然，行尾可能包含 `\n` 與 `\r`，但請始終不要將它們列入考慮 。

## Input

輸入包含多行文字 。每一行文字都被視為一項獨立的輸入 。每行文字的最大長度為 1000 。當遇到檔案結束（EOF）時停止輸入 。

## Output

按照下方格式輸出出現過的 ASCII 字元之 ASCII 數值及其頻率 。每組輸出之間應以一個空行隔開 。請依照字元出現的頻率由小到大（升序）排列輸出 。如果兩個字元的出現頻率相同，則先輸出 ASCII 數值較大的字元資訊 。
