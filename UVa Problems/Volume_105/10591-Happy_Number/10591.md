# 題解

> [!warning]
> - 有很多方法可以「統計是否出現循環」。這裡選擇使用 `std::set`

照題意模擬即可。

## 補充（不需要在程式碼中實現）

探討個位數字的可能性：

- **1** → **1** (Happy!)
- **2** → 4 → 16 → 37 → 58 → 89 → 145 → 42 → 20 → **4**
- **3** → 9 → 81 → 65 → 61 → 37 → ... → **4**
- **4** → 16 → 37 → 58 → 89 → 145 → 42 → 20 → **4** (進入無限循環)
- **5** → 25 → 29 → 85 → 89 → ... → **4**
- **6** → 36 → 45 → 41 → 17 → 50 → 25 → ... → **4**
- **7** → 49 → 97 → 130 → 10 → **1** (Happy!)
- **8** → 64 → 52 → 29 → 85 → 89 → ... → **4**
- **9** → 81 → 65 → 37 → ... → **4**

發現除了 **1** 和 **7** 最終會變成 1 以外，其他的個位數（2, 3, 4, 5, 6, 8, 9）最終都會進入以 **4** 為首的循環：

$$
\{ 4, 16, 37, 58, 89, 145, 42, 20 \}
$$


# Code

```cpp
#include <iostream>
#include <vector>
#include <set>
using namespace std;

set<int> st;

bool Solve(int n) 
{
    if (n == 1)  { return true; }    
    
    int sum = 0;
	while (n > 0)
	{
		sum += (n % 10) * (n % 10);
		n /= 10;
	}
    
    // 檢查是否進入循環
    if (st.count(sum))  { return false; }
    
    st.insert(sum);
    return Solve(sum);
}

int main() 
{
	ios_base::sync_with_stdio(false); cin.tie(nullptr);
	
    int T;  cin >> T;
    for (int t = 1; t <= T; ++t) 
    {
        st.clear();
        int N;  cin >> N;
            
        if (Solve(N))  { cout << "Case #" << t << ": " << N << " is a Happy number.\n"; }
        else           { cout << "Case #" << t << ": " << N << " is an Unhappy number.\n"; }
    }
    return 0;
}
```
