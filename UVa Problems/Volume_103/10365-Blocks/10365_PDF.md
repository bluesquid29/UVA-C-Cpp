# 10365 - Blocks

Donald wishes to send a gift to his new nephew, Fooey. Donald is a bit of a traditionalist, so he has chosen to send a set of $N$ classic baby blocks. Each block is a cube, 1 inch by 1 inch by 1 inch. Donald wants to stack the blocks together into a rectangular solid and wrap them all up in brown paper for shipping.

How much brown paper does Donald need?

---

## Input

The first line of input contains $C$, the number of test cases. For each case there is an additional line containing $N$, the number of blocks to be shipped. $N$ does not exceed 1000.

## Output

Your program should produce one line of output per case, giving the minimal area of paper (in square inches) needed to wrap the blocks when they are stacked together.

---

## Sample Input

```
5
9
10
26
27
100
```

## Sample Output

```
30
34
82
54
130
```

---

# 中文翻譯

Donald 想要送一份禮物給他的新侄子 Fooey 。Donald 有點守舊，所以他選擇送一套包含 $N$ 個經典嬰兒積木的組合 。每個積木都是 1x1x1 英吋的正方體 。Donald 想將這些積木堆疊成一個長方體，並用棕色包裝紙全部包起來以便運送 。

請問 Donald 需要多少包裝紙 ？

## Input

輸入的第一行包含一個整數 $C$，代表測資的筆數 。對於每筆測資，會有一行額外的輸入包含 $N$，即要運送的積木數量 。$N$ 不會超過 1000 。

## Output

對於每筆測資，你的程式應輸出各佔一行的結果，給出將這些積木堆疊在一起時，包裹它們所需的最少包裝紙面積（單位為平方英吋）。
