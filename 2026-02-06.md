## 如何在 C/C++ 得到數學上的模運算

```c
int mod(int a, int n) {
    return ((a % n) + n) % n;
}
```

或者更簡潔的版本：

```c
int mod(int a, int n) {
    int r = a % n;
    return r < 0 ? r + n : r;
}
```

這樣 `mod(-100, 3)` 就會得到 2 了。

## C99/C++11 之前的注意事項

在 C99 和 C++11 之前，負數取餘的行為是**實作定義的**（implementation-defined），不同編譯器可能有不同結果。從 C99/C++11 開始才規範為「結果符號跟被除數相同」。
